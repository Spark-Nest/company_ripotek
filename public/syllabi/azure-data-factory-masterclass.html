Program Overview

Labs: Implement an SCD‑2 dataflow; orchestrate with parameters; add failure/success paths; push logs to Log Analytics.

Module 3 (Weeks 9–12): Patterns, CI/CD & Cost

  • Git integration, ARM templates, release pipelines
  • Cost optimization strategies and governance

Labs: Convert pipeline to metadata‑driven; set up Git integration; deploy via ARM to TEST; analyze and reduce runtime cost.
