Business Intelligence Analyst Program (12 Weeks)

Duration: 12 Weeks   |   Investment: CAD $1,799   |   Level: Beginner   |   Format: Live Online

Program Overview

End‑to‑end BI skills from stakeholder requirements to executive-ready dashboards. Learn modeling, SQL, and visualization using Power BI or Tableau.

Learning Outcomes

  • Facilitate requirements and define measurable KPIs
  • Design dimensional models (star schema) for analytics
  • Write SQL for analysis and data preparation
  • Develop clear, accessible dashboards with best practices
  • Communicate insights and drive decisions with data storytelling

Curriculum

Module 1 (Weeks 1–4): Foundations

  • BI lifecycle, stakeholder analysis, KPI frameworks
  • Data literacy and governance basics
  • SQL essentials: joins, aggregates, windows

Labs: Conduct stakeholder interviews (mock); define KPIs; write SQL queries to validate KPI definitions.

Module 2 (Weeks 5–8): Modeling & ETL

  • Dimensional modeling: facts, dimensions, slowly changing dimensions
  • Data quality and validation checks
  • ETL/ELT concepts; Power Query or equivalent tools

Labs: Design a star schema; implement SCD rules; build a small ETL pipeline to populate the model.

Module 3 (Weeks 9–12): Visualization & Delivery

  • Dashboard design, visual hierarchy, accessibility
  • Interactivity: filters, drill‑through, bookmarks
  • Storytelling and executive presentation techniques

Labs: Redesign an existing dashboard; add drill‑through; present insights to peers for feedback.

Capstone

Deliver a BI solution for a mock client including a dimensional model, SQL transformations, and a multi‑page dashboard with documented KPIs.

  • Integrity: Individual assessments; AI tools allowed with understanding

Outcomes
