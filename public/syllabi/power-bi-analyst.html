Program Overview

  • Build interactive, production-ready Power BI dashboards from scratch

Week-by-Week Syllabus

Module 1: Power BI Foundations (Weeks 1–2)

Week 1: Getting Started with Power BI Desktop

  • Orientation & Power BI Ecosystem
  • Power BI Desktop/Service/Mobile overview
  • When to use Power BI vs Excel vs Tableau
  • Interface tour and first visuals
  • Publish to Power BI Service

Week 2: Power Query & Data Transformation

  • Power Query Editor, applied steps
  • Transformations: merge, append, pivot, unpivot, group
  • Parameters, functions, folder ingestion
  • Best practices: query folding, refresh

Labs: Clean/merge data; build function for multiple files. Mini‑Project 1: Sales Performance Dashboard.

Module 2: Data Modeling (Weeks 3–4)

  • Star schema, fact/dimension design, grain
  • Relationships, cardinality, cross-filter direction
  • Date tables, time intelligence prep, hierarchies
  • Optimization: reducing model size, column cardinality

Labs: Role‑playing dates, date table, optimization. Mini‑Project 2: Customer Analytics Dashboard.

Module 3: DAX (Weeks 5–8)

  • Row vs filter context; CALCULATE; FILTER
  • Time intelligence (YTD/QTD/YoY/MoM, running totals)
  • Variables, SWITCH(TRUE()), iterators (SUMX/AVERAGEX)
  • Virtual tables (SUMMARIZE/ADDCOLUMNS), RANKX, TOPN
  • Performance and readability best practices

Mini‑Project 3: Financial Dashboard with Advanced DAX.

Module 4: Advanced Visualizations (Weeks 9–10)

  • Combo, waterfall, funnel, scatter, maps
  • Custom visuals, tooltips, drill‑through, bookmarks
  • Design best practices, accessibility, mobile layout
  • Storytelling with data and executive presentations

Module 5: Power BI Service & Deployment (Week 11)

  • Workspaces, sharing, Apps
  • Scheduled refresh, gateways, incremental refresh
  • Security: RLS/OLS, permissions and roles

Module 6: Capstone & Certification Prep (Week 12)

  • Capstone kickoff, guidance and presentations
  • PL‑300 exam strategy and practice
