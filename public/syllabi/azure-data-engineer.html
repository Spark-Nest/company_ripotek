Azure Data Engineer Program (24 Weeks)

Duration: 24 Weeks   |   Investment: CAD $3,999   |   Level: Intermediate to Advanced   |   Format: Live online (Tue/Thu/Sun 6–9pm MT)   |   Total Hours: 216   |   Prerequisites: Basic SQL and Python recommended

Program Overview

Become proficient in Azure Data Factory, Synapse Analytics, Databricks, and Lakehouse architectures. Aligned to Microsoft DP‑203; graduates target $95K–$130K roles.

Learning Outcomes

  • Design and implement Azure data architectures (Data Lake, Synapse, Fabric)
  • Build production-grade ELT pipelines (ADF, Databricks)
  • Implement medallion architecture (Bronze/Silver/Gold) with Delta Lake
  • Governance with Purview; security and access controls
  • Performance optimization and cost control
  • Infrastructure as Code (Bicep/Terraform)

Structure

  • Live Sessions: 72 × 3h = 216h
  • Office Hours: 48h
  • Lab Work: 144–216h self-paced
  • Total Time: 12–15h/week

Phase 1: Azure Fundamentals (Weeks 1–4)

Week 1: Azure Basics & Data Storage

  • Cloud concepts, Azure Portal/CLI
  • Resource groups, budgets
  • Storage accounts & ADLS Gen2, ACLs, security

Week 2: SQL Fundamentals

  • T‑SQL basics, joins, window functions
  • Performance and query plans

Week 3: Python for Data Engineering

  • Pandas, APIs, file I/O, Azure SDK

Week 4: Azure SQL & Synapse Serverless

  • Synapse workspace, external tables, OPENROWSET, CETAS

Phase 2: ADF & ELT Pipelines (Weeks 5–10)

  • Copy activity, schema drift, performance tuning
  • Mapping Dataflows: transforms, joins, aggregations
  • SCD Type‑2 with alter row
  • Triggers, parameters, variables, dynamic content
  • Error handling, monitoring, alerting
  • Incremental loads, CDC, merge/upsert
  • CI/CD: Git integration, ARM templates, Azure DevOps

Phase 3: Databricks & Medallion (Weeks 11–16)

  • Spark internals, clusters, PySpark DataFrames
  • Delta Lake fundamentals: ACID, time travel, OPTIMIZE, VACUUM
  • Bronze/Silver/Gold design, Auto Loader, quality, SCD in Delta
  • Aggregation for Gold, star schema, performance (Z‑ORDER, caching, broadcast)
  • Workflows/Jobs: DAGs, parameters, alerts

Phase 4: Advanced Topics (Weeks 17–20)

  • Streaming with Event Hubs + Structured Streaming
  • Synapse Spark: pools, integration with SQL pools
  • Purview: catalog, classification, lineage, access policies
  • IaC with Bicep/Terraform, multi‑env pipelines

Phase 5: Capstone & Certification (Weeks 21–24)

  • End‑to‑end project: multi‑source ingestion, medallion, 5+ ELT pipelines, governance, IaC
  • DP‑203 prep: practice exam, deep‑dives, exam strategy
  • Presentations, career prep
