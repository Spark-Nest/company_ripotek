AI Engineer Program (24 Weeks)

Duration: 24 Weeks   |   Investment: CAD $3,999   |   Level: Advanced   |   Format: Hybrid

Program Overview

Build production‑ready AI solutions with Azure OpenAI and robust MLOps. Implement RAG systems, agentic workflows, and responsible AI practices. Aligned to Microsoft AI‑102.

Curriculum

Phase 1 (Weeks 1–6): GenAI & Prompt Engineering

  • LLM fundamentals; models, tokens, safety
  • Prompt patterns; few‑shot and chain‑of‑thought
  • Azure OpenAI API; grounding and guardrails

Labs: Implement system/user prompts; build few‑shot examples; add safety filters; measure output quality with simple evals.

Phase 2 (Weeks 7–12): RAG & LangChain

  • Embeddings, chunking, retrieval strategies
  • Vector DBs (e.g., Azure AI Search)
  • LangChain workflows and agents

Labs: Index a small corpus; build a RAG chain with re‑ranking; add an agent with tools; latency/cost profiling.

Phase 3 (Weeks 13–18): Azure ML & Fine‑tuning

  • Azure ML pipelines and model registry
  • Fine‑tuning workflows and evaluation
  • Responsible AI: bias, privacy, monitoring

Labs: Create an AML pipeline; register a model; fine‑tune on a small dataset; evaluate with custom metrics and human review.

Phase 4 (Weeks 19–24): MLOps & Production

  • CI/CD for ML; testing and observability
  • Serving, scaling, and cost management
  • Capstone build and deployment

Labs: Add CI/CD checks; deploy an endpoint; integrate telemetry and guardrails; A/B test prompts or models.

Key Topics

  • Azure OpenAI and prompt engineering
  • RAG with vector databases
  • LangChain and agentic systems
  • Fine‑tuning and Azure ML
  • MLOps CI/CD; Responsible AI

Assessment & Certification

  • Capstone: Design and deploy a production RAG/agent system
  • Certification: Prepares for AI‑102 (Azure AI Engineer Associate)

Policies

  • Attendance 80% minimum; recordings available
  • Refunds: Full (minus $200) before Week 2; 50% Weeks 2–4; none after Week 4
  • Academic integrity: individual assessments; AI tools allowed with understanding
