Program Overview

Learn Python fundamentals, data manipulation with pandas/NumPy, and practical data workflows including APIs, cleaning, and automation.

Curriculum

Module 1 (Weeks 1–4): Core Python

  • Syntax, control flow, functions, error handling
  • Data structures: lists, dicts, sets, tuples
  • Working with files and environments

Labs: Build CLI utilities; parse files; handle errors gracefully; package a small utility.

Module 2 (Weeks 5–8): pandas & NumPy

  • DataFrames and Series; filtering, grouping, joins
  • Missing data, reshaping, time series
  • Numerical ops with NumPy

Labs: Clean messy CSVs; join datasets; compute KPIs; time‑series resampling.

Module 3 (Weeks 9–12): Visualization & Automation

  • Matplotlib/Seaborn for plots
  • Calling APIs, parsing JSON, basic scraping
  • Building automated data scripts and notebooks

Labs: Build API ingestion script; cache responses; generate daily report notebook with charts.

Capstone

Design an automated data pipeline using an external data source, cleaning/transformation, and a final analysis/visualization deliverable.
